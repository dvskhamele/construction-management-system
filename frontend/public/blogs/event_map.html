Event Map

ToolOpened

A user opened a tool.

tool_slug: (string) Slug of the tool opened

timestamp: (datetime) Time the tool was opened

user_agent: (string) User agent of the client

ip_address: (string) IP address of the client

ToolSubmitted

A user submitted a tool form.

tool_slug: (string) Slug of the tool submitted

timestamp: (datetime) Time the tool form was submitted

input_data: (object) Anonymized input data for the tool

user_agent: (string) User agent of the client

ip_address: (string) IP address of the client

ToolResultViewed

A user viewed the results of a tool.

tool_slug: (string) Slug of the tool whose results were viewed

timestamp: (datetime) Time the tool results were viewed

result_id: (string) Unique ID of the generated result

user_agent: (string) User agent of the client

ip_address: (string) IP address of the client

LeadCaptured

A lead was captured through a tool.

tool_slug: (string) Slug of the tool that captured the lead

timestamp: (datetime) Time the lead was captured

email: (string) Hashed email address of the lead

lead_source: (string) Source of the lead (e.g., tool, homepage)

user_agent: (string) User agent of the client

ip_address: (string) IP address of the client

ErrorEvent

An error occurred during tool execution or page rendering.

error_code: (string) Unique error code

message: (string) Error message

severity: (string) Severity of the error (e.g., low, medium, high)

timestamp: (datetime) Time the error occurred

context: (object) Additional context about the error (e.g., tool_slug, page_path)

user_agent: (string) User agent of the client

ip_address: (string) IP address of the client
