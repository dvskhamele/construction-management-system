What is Schedule Variance?

Schedule Variance (SV) is a critical metric in Earned Value Management (EVM) that indicates whether a project is ahead or behind its planned schedule. It is calculated as the difference between the Earned Value (EV) and the Planned Value (PV).

  • Earned Value (EV): The budgeted cost of the work actually performed.
  • Planned Value (PV): The budgeted cost of the work scheduled to be completed by a given date.
